# Sinatra Project - Social Recipes

Welcome to Social Recipes

## Installation

Copy all folders and files

And then execute:

    $ bundle install

Migrate databases and seed:

    $ rake db:migrate
    $ rake db:seed

## Usage

Launch the server by running:

    $ shotgun
